# Детекция падения человека на видео

## Описание проекта
Разработка системы компьютерного зрения для детекции падения человека по видеопоследовательности.

## Структура проекта
- `models/` - реализации архитектур нейронных сетей
- `train_all_models.py` - скрипт для обучения и сравнения моделей
- `prepared_data/` - подготовленные данные (не включены в репозиторий)
- "https://github.com/ekramalam/GMDCSA24-A-Dataset-for-Human-Fall-Detection-in-Videos" - использовался репозиторий для датасета

## Запуск
1. Установи зависимости: `pip install -r requirements.txt`
2. Подготовь данные (см. Приложение №1 в отчете)
3. Запусти обучение: `python train_all_models.py`

## Результаты
- CNN + LSTM: F1 = 0.495
- 3D ResNet: F1 = 0.639
- SlowFast: неудачное обучение
- TimesFormer: неудачное обучение
- VideoMae: неудачное обучение
- MediaPipe + LSTM: планируется
